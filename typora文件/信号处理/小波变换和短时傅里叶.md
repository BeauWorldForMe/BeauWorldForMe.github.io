# Gabor变换

+ 图像的Gabor变换
+ 也是一种小波形式
+ why？
  + 经典傅里叶变换只能反映信号的整体特征（时域频域）
  + 短时傅里叶变换，不同窗形式具有不同的时间和频率分辨能力
  + Gabor变换是短时傅里叶变换中当窗函数取为高斯函数时的一种特殊情况
    + 高斯窗时 不确定度的概念
+ 和STFFT短时傅里叶的类比
+ 傅里叶变化的基是三角函数，是确定的，不用选母函数 h(t)
+ 在图像分析方面的应用
  + 图像的频域和时域代表什么
    + 一维：在时间和频率之间做了栅格
    + 二维：在图像x和y之间做了栅格（时频--->空间）
